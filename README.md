# 📱 Curiosidades de Tecnologia

Projeto de site sobre a **história do mascote do Android**, desenvolvido com **HTML e CSS**.

## 📚 Sobre o projeto

Este projeto foi criado como prática de desenvolvimento web, focando em:

* Estrutura semântica com HTML
* Estilização com CSS
* Layout responsivo
* Uso de imagens adaptáveis
* Incorporação de vídeo

## 🛠️ Tecnologias usadas

* HTML5
* CSS3
